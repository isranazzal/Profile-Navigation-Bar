# Profile Navigation Bar

## Description

A simple frontend project that demonstrates the creation of a profile navigation bar using HTML5 and CSS3. The project focuses on designing a clean user interface with navigation links, profile options, and a styled sign-out button.

## Technologies Used

- HTML5
- CSS3

## Features

- Custom navigation bar design
- Profile navigation links
- Styled active link
- Sign-out button component
- Clean and organized layout
